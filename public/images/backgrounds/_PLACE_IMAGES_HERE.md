# Фоны страниц — сюда класть картинки

Каждая страница тянет фон из `public/images/backgrounds/{имя}.jpg`.
Если файла нет — страница просто остаётся на тёмном фоне, ошибки не будет.

Нужные файлы (формат **.jpg**, горизонтальные, рекомендую **1920×1080** или больше):

| Файл | Страница | Что на картинке |
|---|---|---|
| `landing.jpg` | Главный экран `/` | Эпичный вид Темнодушного Лета — тёмный славянский пейзаж, врата в мир, луна |
| `select-class.jpg` | Выбор класса | Зал Судьбы — 6 силуэтов/масок классов в полумраке, свечи |
| `profile.jpg` | Профиль (свой и чужой `/player/[id]`) | Личная келья путника — стол с картой, доспех на стойке, тусклый очаг |
| `inventory.jpg` | Инвентарь | Старый сундук, полки с зельями и артефактами, мешковина |
| `quests.jpg` | Свитки квестов | Стол квестодателя — пергаментные свитки, перо, чернильница, восковые печати |
| `feed.jpg` | Лента/Хроника | Таверна с доской объявлений, или гобелен с лидербордом |
| `duels.jpg` | Арена дуэлей | Кровавая арена, скрещённые мечи, факелы, зрительские трибуны в тумане |
| `shop.jpg` | Лавка Романала | Тесная торговая лавка — полки с зельями, свитками, монеты, весы, хитрый торговец |
| `map.jpg` | Карта мира | Тёмный пергаментный стол картографа, свеча, мгла по краям — фон под картой мира |

Стиль для всех — **тёмное славянское фэнтези, Dark Souls × деревня Гадюкино**:
приглушённые цвета, туман, золото свечей, ощущение древности и лёгкого кринжа.
Картинки видны приглушённо (~35% яркости) под тёмной вуалью — не делай их слишком
светлыми, иначе текст поверх будет плохо читаться.

Midjourney-промпты для этих фонов будут в `PROMPTS.md` в корне проекта.

После добавления файлов этот `.md` можно удалить.
