# Звук — сюда класть аудиофайлы

Все файлы — **одной папкой** `public/audio/`, формат **.mp3**.
Файлов может не быть: тогда просто тихо, без ошибок.
Управление — кнопки в правом нижнем углу сайта (🎵 музыка / 🔊 звуки).

## Музыка

- `track1.mp3` — фоновый трек (зацикливается).

Хочешь несколько треков — залей `track2.mp3`, `track3.mp3` и т.д. и впиши их
в массив `MUSIC_TRACKS` в `lib/sound.ts`. Музыка по умолчанию выключена
(браузеры блокируют автоплей) — включается кнопкой 🎵.

## Звуки событий

Короткие клипы (0.3–2 сек), кладём туда же — `public/audio/`:

| Файл | Когда играет |
|---|---|
| `click.mp3` | клик по кнопке |
| `move.mp3` | переход на локацию |
| `roll.mp3` | ролл игры |
| `wheel.mp3` | крутка колеса предметов |
| `quest_complete.mp3` | выполнен квест |
| `complete.mp3` | засчитана игра |
| `coin.mp3` | покупка / продажа в Лавке |
| `fail.mp3` | дроп / провал / ловушка |
| `levelup.mp3` | новый уровень |

Подключены в коде: `move`, `complete`, `fail`, `coin`, `click`.
