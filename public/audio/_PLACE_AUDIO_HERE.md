# Звук — сюда класть аудиофайлы

Формат — **.mp3**. Файлов может не быть: тогда просто тихо, без ошибок.
Управление звуком — кнопки в правом нижнем углу сайта (🎵 музыка / 🔊 звуки).

## Музыка — `public/audio/music/`

Фоновый плейлист, играет по кругу. Имена файлов:

- `track1.mp3`
- `track2.mp3`
- `track3.mp3`

Хочешь больше треков — добавь файлы и впиши их в `MUSIC_TRACKS` в `lib/sound.ts`.
Музыка по умолчанию **выключена** (браузеры блокируют автоплей) — включается кнопкой 🎵.

## Звуки событий — `public/audio/sfx/`

Короткие клипы (0.3–2 сек):

| Файл | Когда играет |
|---|---|
| `click.mp3` | клик по кнопке |
| `move.mp3` | переход на локацию |
| `roll.mp3` | ролл игры |
| `wheel.mp3` | крутка колеса предметов |
| `quest_complete.mp3` | выполнен квест |
| `complete.mp3` | засчитана игра |
| `coin.mp3` | покупка / продажа в Лавке |
| `fail.mp3` | дроп / провал / попал в ловушку |
| `levelup.mp3` | новый уровень |

Уже подключены в коде: `move`, `complete`, `fail`, `coin`, `click`.
Остальные (`roll`, `wheel`, `quest_complete`, `levelup`) — ключи готовы, вызов
`playSfx("...")` можно добавить в нужный момент (скажи — подключу).

После добавления файлов этот `.md` можно удалить.
